The test dataset

The data set initially loaded is sourced from "OzDASL" the "Australasian Data and Story Library", and consists of set of observations related to a subjects pulse rate before and after strenuous exercise. Various aspects of the subjects life style are recorded along with the data related to the pulse readings. The first few rows of the data are shown below.

The central question...

The main question that this dataset could be used to answer is by how much a subjects pulse rate changes with exercise and what lifestyle factor affect this change.
